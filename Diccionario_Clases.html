public Map<String,Object> procesarActividad(Long usuarioId, String tipoActividad) Busca usuario (RuntimeException si no). Switch tipoActividad:
• TAREA_CREADA: +10 pts; si totalTareas==1 → otorgarLogro("PRIMERA_TAREA")
• TAREA_COMPLETADA: +20 pts; si completadas≥5 → CINCO_TAREAS; si ≥10 → DIEZ_TAREAS
• QUIZ_COMPLETADO: +50 pts; si quizzes==1 → PRIMER_QUIZ; si ≥4 → TODOS_BASICO; si ≥8 → TODOS_AVANZADO
• NIVEL_AVANZADO: otorgarLogro("NIVEL_AVANZADO")
Suma puntos a usuario, guarda. Retorna {puntosGanados, puntosTotal, nuevosLogros}.
private List<Map<String,Object>> otorgarLogro(Usuario usuario, String codigo) Busca logro por código. Si no existe o el usuario ya lo tiene, retorna lista vacía. Crea LogroUsuario (usuarioId, logro, obtenidoAt=now), guarda. Suma puntosRecompensa al usuario. Retorna [{codigo, nombre, icono, puntos}].
