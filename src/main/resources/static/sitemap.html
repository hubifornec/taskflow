API REST — Endpoints del backend
Método Endpoint Descripción
🔐 Autenticación
POST /api/auth/register Registro de nuevo usuario
POST /api/auth/login Inicio de sesión
👤 Usuarios
GET /usuarios/{id} Obtener datos de un usuario
POST /usuarios/{id}/completar-quiz Registrar quiz completado y actualizar nivel
📋 Tareas
GET /tareas?usuarioId={id} Listar tareas de un usuario
POST /tareas Crear nueva tarea
PUT /tareas/{id} Actualizar tarea existente
DELETE /tareas/{id} Eliminar tarea
🎓 Quizzes
GET /api/quiz Listar todos los cuestionarios
GET /api/quiz/nivel/{nivel} Cuestionarios filtrados por nivel
GET /api/quiz/{id} Preguntas de un cuestionario
POST /api/quiz/validar/{preguntaId} Validar respuesta y obtener explicación
GET /api/teoria/{cuestionarioId} Contenido teórico de un módulo
🚀 Simulación de Sprint
POST /api/simulacion/iniciar Iniciar nueva simulación de sprint
GET /api/simulacion/activa/{usuarioId} Obtener simulación activa del usuario
PATCH /api/simulacion/item/{id}/toggle Mover tarea entre backlog y sprint
PATCH /api/simulacion/item/{id}/completar Marcar tarea de simulación como completada
POST /api/simulacion/cerrar/{id} Cerrar sprint y obtener retrospectiva
📦 Exportación SCORM
GET /api/scorm/app Descarga la app completa como paquete SCORM 1.2
GET /api/scorm/quiz/{id} Descarga un cuestionario individual como SCORM 1.2
